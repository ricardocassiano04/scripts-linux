# scripts-linux

## Vários scripts para Linux.

Scripts que utilizo no dia a dia para facilitar várias tarefas repetitivas. 


### Scripts Comuns

* [adjust-home-permission](https://github.com/ricardocassiano04/scripts-linux/blob/main/comun/adjust-home-permission.sh)  - Script para ajustar as permissões da pasta $HOME e alguns tipos de arquivo. Ele remove a permissão de leitura para outros.
* [backup-arquivos-home](https://github.com/ricardocassiano04/scripts-linux/blob/main/comun/backup-arquivos-home.sh)  - Script de backup de pastas da $HOME para um dispositivo externo.
* [hblock-install](https://github.com/ricardocassiano04/scripts-linux/blob/main/comun/hblock-install.sh) - Instala [Hblock](https://github.com/hectorm/hblock) para bloquear propagandas e malwares.
* [install-mint-theme](https://github.com/ricardocassiano04/scripts-linux/blob/main/comun/install-mint-theme.sh)  - Instala ou atualiza os temas do Linux Mint.
* [linux-adjustments](https://github.com/ricardocassiano04/scripts-linux/blob/main/comun/linux-adjustments.sh)  - Instala ou atualiza os temas do Linux Mint direto do Github.
* [npm-change-default-install-directory-to-home](https://github.com/ricardocassiano04/scripts-linux/blob/main/comun/npm-change-default-install-directory-to-home.sh) - Altera a pasta padrão dos pacotes npm para $HOME/.npm-packages



### Debian

* [configure-eclipse-temurim-java-lts-debian](https://github.com/ricardocassiano04/scripts-linux/blob/main/debian/configure-eclipse-temurim-java-lts-debian.sh) - Instala o Java OpenJDK do [Eclipse Temurin](https://adoptium.net/) no Debian 12+ / Ubuntu 24.04+.
* [debian-development-packages](https://github.com/ricardocassiano04/scripts-linux/blob/main/debian/debian-development-packages.sh) - Instala alguns pacotes para desenvolvemento python no Debian/Ubuntu.
* [debian-initial-setup](https://github.com/ricardocassiano04/scripts-linux/blob/main/debian/debian-initial-setup.sh) - Configurações iniciais do Linux Debian 12 Bookworm.

### Linux Mint

* [linux-mint-initial-setup](https://github.com/ricardocassiano04/scripts-linux/blob/main/linux-mint/linux-mint-initial-setup.sh) - Configurações iniciais do Linux Mint 22+.
